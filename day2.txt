명명법

카멜케이스 (camel case) : 기본적으로 소문자로 작성하되 단어간 구분을 위해 첫단어를 제외한 나머지 단어의 첫자를 대문자로 하는 것.
 
 - 카멜 케이스는 자바의 멤버 변수 지역변수 등의 명명에 사용된다. ex) String some StringName = ""
							  int someIntName = 0;
파스칼케이스 (pascal case) : 기본적으로 소문자로 작성하되, 모든 단어의 첫자를 대문자로 작성하는것.
 - 파스칼 케이스는 자바의 클래스 인터페이스 열거형 등의 명명에 사용한다. ex) public class SomeClassName {...}
								    public interface SomeClassName {...}
								    public enum SomeClassName {...}
스네이크 케이스 (snake case) : 전체 소문자로 작성하되 단어간 구분을 언더스코어(_)로 하는것
 - 스네이크 케이스는 일부 관계형 데이터베이스(RDBMS)의 구성요소 및 PHP 변수 명명에 사용한다. ex) CREATE SCHEMA 'some_schema_name';
										     CREATE TABLE 'some_schema_name'. 'some_table_name' (. . . );
대문자 스네이크 케이스 (upper snake case) : 전체 대문자로 작성하되 단어간 구분을 언더스코어(_)로 하는것
 - 대문자 스네이크 케이스는 자바에서 정적이고 리터럴인 상수들을 명명에 사용한다 . ex) public static final  double PI  = 3.141592D;
									 public static final  double DAYS_IN_YEAR = 365.2425D;
케밥 케이스 (kebeb case) : 전체 소문자로 작성하되, 단어간 구분을 대쉬(-)로 하는것.
 - 케밥 케이스는 HTML 대부분의 명명(태그,속성 등)에 사용된다.  ex) <li class = "some -class- name" data-some-message-text = "HELLO!"> </li>

HTML은 기본적으로 대소문자를 가리지 않는다.

ex) <head>
		<TITLE>제목입니다.</TITLE>
     <head>    
	가능은 하나 되도록 소문자로 통일하여 작성한다.

a
 - a (Anchor)태그는 현재 페이지에서 다른 페이지로 이동할 수 있는 링크를 만들기 위해 사용한다.
 - 자동으로 닫는 태그 : X
 - 시멘틱(의미를 부여한 태그) : O 
 - 속성
     - href : 이동할 주소 및 경로
     - target : 이동할 방식
	- _blank : 새로운 탭에서 이동
	- _self : 현재탭(창)에서 바로 이동

b
 - b (Bold )태그는 텍스트를 굵게 만든다.
 - 자동으로 닫는 태그 : X
 - 시멘틱(의미를 부여한 태그) : X
 - 단순히 텍스트를 굵게 만들기 위해서는 <b> 태그가 아닌 CSS의 font - weight 속성을 사용하여야한다.




strong
 - strong 태그는 텍스트를 굵게 만든다.
 - 자동으로 닫는 태그 : X
 - 시멘틱(의미를 부여한 태그) : O
 - <strong>  태그는 아래와 같은 시멘틱을 가진다.
	- 검색 엔진 최적화(SEO, Search Engine Optimization)가 이루어져 검색엔진으로 하여금 해당 태그가 가지고 있는 내용이 문서에서
 	   중요한 내용임을 알린다.
	- 스크린 리더(Screen Reader) 로 하여금 해당 태그가 가지고 있는 내용을 보다 강한 어조로 
	  읽도록 하여 웹 접근성(Web Accessibilty)을 향상시킨다.

br
 - br(Break)태그는 개행을 위해 사용한다.
 - 자동으로 닫는 태그 : O
 - 시멘틱(의미를 부여한 태그) : X
 - 문서 레이아웃(디자인)을 목적으로 사용하여서는 안 된다. 즉, 전적으로 텍스트에서 사용한다.


* Paragraph 문단


h1, h2, h3, h4, h5, h6

 - h1, h2, h3, h4, h5, h6 (Heading)태그는 이를 포함하는 태그의 제목이나 간략한 주제를 나타내기 위해 사용한다.
 - 자동으로 닫는 태그 : X 
 - 시멘틱(의미를 부여한 태그) : O
 - 숫자가 작을수록(1에 가까울수록)의미가 큰 태그이다.
 - <h1> 태그는 주로 문서의 제목과 동일하게 지정하며 문서에 하나만 존재하는편이다.
 - 동일한 단계 안에 동일한 등급의 헤딩 태그를 두지 않는 것이 좋다.
 

cite
 - cite태그는 텍스트를 기울이기 위해 사용한다.
 - 자동으로 닫는 태그 : X 
 - 시멘틱(의미를 부여한 태그) : O
 - 단순히 텍스트를 기울이기 위해서  <cite> 태그가 아닌  CSS의 font - style 속성을 사용하여야 한다.
 - 출판물, 영화 등 저작물의 출처나 제목 등을 표시하기위해 사용한다.


i
- i(Italic)태그는 텍스트를 기울이기 위해 사용한다.
 - 자동으로 닫는 태그 : X 
 - 시멘틱(의미를 부여한 태그) : O
 - 단순히 텍스트를 기울이기 위해서  <i> 태그가 아닌  CSS의 font - style 속성을 사용하여야 한다.
 - 본문의 주된 언어와 다른 언어를 담기 위해 사용한다.



em
- em(Emphasize)태그는 텍스트를 기울이기 위해 사용한다.
 - 자동으로 닫는 태그 : X 
 - 시멘틱(의미를 부여한 태그) : O
 - 단순히 텍스트를 기울이기 위해서 태그가 아닌 <em> CSS의 font - style 속성을 사용하여야 한다.
 - 어떤 문장이나 문장의 부분을 강조하기 위해 사용한다.
 

img
 - img (Image) 태그는 이미지를 삽입하기위해 사용한다.
 - 자동으로 닫는 태그 : O
 - 시멘틱(의미를 부여한 태그) : O
 - 속성
 	- alt : src 속성에 명시된 이미지를 표시할 수 없을때 대신(Alternative)하여 표시하거나 스크린 리더가 읽을 텍스트 문자열이다.
	        해당 속성을 명시하지 않는 것은 HTML 5 표준 위반이다. 
	- src : 표시할 이미지의 주소, 경로 혹은64진법(Base64)데이터 문자열이다.



* 2 진법 : 0, 1
  8 진법 : 0~7
  10 진법 : 0~9
  16 진법 : 0~9, A~F
  64 진법 : 0~9, 영어 대문자(A ~ Z), 영어 소문자(a ~ z), (+ , /) 이다.


li
 - li (List Item) 태그는 목록의 항목 하나를 가지기 위한 태그이다.
 - 자동으로 닫는 태그 : X
 - 시멘틱(의미를 부여한 태그) : O
  - 부모로 <ol> , <ul> 태그 만 가질수 있다.

ol
 - ol (Ordered List) 태그는 순서와 유관한 항목을 가지기 위한 목록 태그이다.
 - 자동으로 닫는 태그 : X
 - 시멘틱(의미를 부여한 태그) : O
 - 자식으로 <li> 태그 만 가질수 있다.


ul
 - ul (Unordered List) 태그는 순서와 무관한 서로 관련있는 항목을 가지기 위한 목록 태그이다.
 - 자동으로 닫는 태그 : X
 - 시멘틱(의미를 부여한 태그) : O
 - 자식으로 <li> 태그 만 가질수 있다.

밑으로 복사 shift + alt + 아래
ctrl alt 

u
 - u(Underline)태그는 오자를 강조하기 위해 사용하며 시각적 효과로 밑줄이 그인다.
 - 자동으로 닫는 태그 : X
 - 시멘틱(의미를 부여한 태그) : O
 - 단순히 밑줄을 긋기 위해서는 <u>태그가 아닌 CSS의 text - decoration 속성을 사용하여야한다.

sub
 - sub(Subscript) 태그는 아래 첨자를 위해 사용하는 태그이다.
 - 자동으로 닫는 태그 : X
 - 시멘틱(의미를 부여한 태그) : X

sup
  - sup(Superscript) 태그는 윗 첨자를 위해 사용하는 태그이다.
 - 자동으로 닫는 태그 : X
 - 시멘틱(의미를 부여한 태그) : X

href 값은 # 이다.
&lt; => 여는 꺽쇠 괄호 (Less Than, 미만)
&gt; => 닫는 꺽쇠 괄호 (Gather Than, 초과)
&nbsp; => 띄우기
